## 2. 타겟 사용자

### 주 타겟: 병원 담당자 (B2B)
- **병원 마케팅 담당자**: 해외 환자 유치 채널을 찾고 있는 실무자
- **국제진료센터 팀장**: 외국인 환자 인프라를 구축하려는 의사결정자
- **원장/경영진**: ROI 중심으로 파트너십을 검토하는 최종 결정자

### 사용자 시나리오
> 병원 마케팅 담당자가 해외 환자 유치 에이전시를 검색하거나, 
> WITHUS MEDICARE 영업 담당자에게 명함/링크를 받고 사이트에 접속한다.
> 사이트를 스크롤하며 서비스 내용과 AI 콘텐츠 샘플을 확인하고,
> "문의해볼 만하다"고 판단하여 연락을 취한다.

### 사이트 목표
**핵심 전환 액션**: 문의하기 (전화 / 이메일 / 문의 폼)

## 3. 사이트 구조 (섹션 구성)

원페이지 스크롤 구조로, 상단 네비게이션에서 각 섹션으로 앵커 이동.

### 3.1 헤더 (고정 네비게이션)
- 로고 (좌측)
- 네비게이션 메뉴: 서비스 | AI 콘텐츠 | 프로세스 | 회사소개 | 문의하기
- CTA 버튼: "문의하기" (우측, 강조 색상)
- 스크롤 시 배경 불투명 전환 (glassmorphism 또는 solid)

### 3.2 히어로 섹션
**목적**: 첫 화면에서 "이 회사가 뭘 하는 곳인지" 즉시 전달

- **헤드라인**: 병원 담당자에게 직접 말하는 톤
  - 예시: "AI 콘텐츠로 해외 환자를 연결합니다"
  - 예시: "병원의 해외 환자 유치, AI로 시작하세요"
- **서브 카피**: 1~2문장으로 핵심 가치 제안
  - 예시: "AI 기반 숏폼 콘텐츠 제작부터 환자 상담, 통역까지. 해외 환자 유치의 전 과정을 함께합니다."
- **CTA 버튼**: "서비스 알아보기" / "무료 상담 신청"
- **비주얼**: 풀스크린 배경 영상 또는 고급 이미지 (의료 + 글로벌 느낌)

### 3.3 문제 제기 섹션 (Pain Points)
**목적**: 병원 담당자의 고민을 짚어 공감 형성

3~4개의 pain point를 카드 또는 아이콘 기반으로 제시:

1. **"해외 마케팅, 어디서부터 시작해야 할지 모르겠다"**
   - 채널 선정, 언어, 현지 트렌드 등 진입 장벽
2. **"콘텐츠를 만들 전문 인력이 없다"**
   - 외국어 영상, SNS 운영 인력 부재
3. **"에이전시에 맡겼는데 성과가 안 보인다"**
   - 기존 에이전시의 일회성 콘텐츠 납품 방식의 한계
4. **"외국인 환자 응대 인프라가 부족하다"**
   - 통역, 상담, 사후관리 등 운영 역량 부재

### 3.4 서비스 소개 섹션
**목적**: WITHUS MEDICARE가 제공하는 서비스를 구체적으로 설명

각 서비스를 카드/탭 형식으로 구성. 나열이 아닌 **"무엇을 해주는가"** 중심.

#### 서비스 1: AI 기반 SNS 콘텐츠 제작
- AI 숏폼 영상 제작 (병원 소개, 시술 안내, 후기 등)
- AI 모델을 활용한 병원 홍보 콘텐츠
- 타겟 국가/언어별 맞춤 콘텐츠 현지화
- Instagram, YouTube Shorts, TikTok 등 채널 운영

#### 서비스 2: 외국인 환자 유치 마케팅
- 타겟 국가별 SNS 광고 운영
- 인플루언서 마케팅 (현지 인플루언서 섭외 및 협업)
- 검색 최적화 및 온라인 노출 전략

#### 서비스 3: 환자 커뮤니케이션 & 컨시어지
- 다국어 상담 대응 (DM, 전화, 메신저)
- 의료 통역 서비스 (내원 시 동행)
- 비자, 항공, 숙소 등 입국 지원
- 귀국 후 사후관리 및 팔로업

### 3.5 AI 콘텐츠 쇼케이스 섹션 ⭐ (킬링 섹션)
**목적**: "이런 퀄리티의 콘텐츠를 만들 수 있다"를 직접 보여주는 핵심 영업 자산

- **섹션 타이틀**: "AI가 만드는 병원 콘텐츠, 직접 확인하세요"
- 숏폼 영상 샘플 2~4개 임베드 (YouTube/Instagram 임베드 또는 자체 호스팅)
  - 예: AI 모델 병원 소개 영상
  - 예: AI 생성 시술 안내 숏폼
  - 예: before/after 스타일 콘텐츠
- 각 영상 하단에 간단한 설명 (제작 방식, 활용 채널 등)
- 갤러리/캐러셀 형태로 구성
- **콘텐츠 소스**: 강남좋은눈안과 관련 AI 콘텐츠 포트폴리오 활용

### 3.6 협업 프로세스 섹션
**목적**: "문의하면 실제로 어떻게 진행되는지" 투명하게 보여주기

타임라인 또는 스텝 형태로 시각화:

1. **상담 & 니즈 파악** → 병원 현황 및 목표 분석
2. **전략 수립** → 타겟 국가, 채널, 콘텐츠 방향 설계
3. **콘텐츠 제작** → AI 기반 숏폼/SNS 콘텐츠 제작
4. **채널 운영 & 광고** → SNS 운영, 광고 집행, 인플루언서 협업

### 3.7 회사 소개 섹션
**목적**: 신뢰감 형성

- 회사명: WITHUS MEDICARE
- 비전/미션 간결하게 (2~3문장)
- 외국인환자유치업 등록 정보 (등록번호 표시 → 합법성/신뢰)
- 대표 또는 팀 소개 (선택사항 — 얼굴이 있으면 신뢰도 상승)

### 3.8 문의하기 섹션 (CTA)
**목적**: 전환 — 실제 문의 액션 유도

- **헤드라인**: "해외 환자 유치, 지금 시작하세요"
- 문의 폼:
  - 병원명
  - 담당자명
  - 연락처 (전화번호)
  - 이메일
  - 관심 서비스 (체크박스: SNS 콘텐츠 / 환자 유치 / 통역 서비스 / 기타)
  - 문의 내용 (자유 입력)
- 직접 연락처 병기:
  - 📞 010-2074-0000
  - 📧 Withusmedicare@gmail.com
- 카카오톡 채널 링크 (있는 경우)

### 3.9 푸터
- 회사 정보 (상호, 대표, 사업자등록번호, 외국인환자유치업 등록번호)
- 연락처
- Copyright
- 개인정보처리방침 링크 (문의 폼 수집 시 필수)

## 4. 디자인 가이드라인

### 4.1 전체 톤
- **프리미엄 + 신뢰**: 다크 베이스 (Deep Navy 또는 Charcoal) + 골드/화이트 포인트
- 의료 업계 특유의 "깨끗한" 느낌보다는 **비즈니스 파트너**로서의 세련됨 강조
- 과도한 장식 배제, 여백과 타이포그래피로 고급감 연출

### 4.2 컬러 팔레트 (안)
| 용도 | 컬러 | 비고 |
|------|------|------|
| Primary Dark | #0A1628 (Deep Navy) | 배경, 히어로 |
| Primary Accent | #C8A96E (Muted Gold) | CTA, 강조 요소 |
| Text Primary | #FFFFFF | 다크 배경 위 본문 |
| Text Secondary | #B0B8C4 | 서브 카피, 설명 |
| Section Alt BG | #F5F5F7 (Light Gray) | 밝은 섹션 배경 |
| Success/Trust | #2E8B6E (Emerald) | 신뢰 관련 요소 |

### 4.3 타이포그래피
- 한글: Pretendard 또는 SUIT (가독성 + 모던)
- 영문: Inter 또는 Poppins
- 히어로 헤드라인: Bold, 48~64px
- 섹션 타이틀: SemiBold, 32~40px
- 본문: Regular, 16~18px, line-height 1.6~1.8

### 4.4 레이아웃
- 최대 너비: 1200px (컨텐츠 영역)
- 히어로: 풀스크린 (100vh)
- 섹션 간 충분한 여백 (padding: 120px 0 이상)
- 모바일 반응형 필수 (병원 담당자가 모바일로 볼 가능성 높음)

### 4.5 인터랙션
- 스크롤 기반 fade-in 애니메이션 (과하지 않게)
- 숏폼 영상 hover 시 자동 재생 (muted)
- 네비게이션 smooth scroll
- 프로세스 섹션: 스크롤 연동 스텝 하이라이트

## 5. 기술 요구사항

### 5.1 프론트엔드
- **Framework**: React (Next.js 권장 — SEO, 이미지 최적화)
- **스타일링**: Tailwind CSS
- **애니메이션**: Framer Motion
- **영상 임베드**: YouTube/Instagram iframe 또는 자체 호스팅 (mp4)
- **폼 처리**: 
  - 옵션 A: Formspree / Getform (간편, 무료 티어)
  - 옵션 B: n8n webhook → Google Sheets + 이메일 알림 (자체 구축)
  - 옵션 C: Resend API (이메일 직접 발송)

### 5.2 배포
- **호스팅**: Vercel (무료 티어로 충분)
- **도메인**: 별도 구매 필요 (예: withusmedicare.co.kr 또는 withusmedicare.com)
- **SSL**: Vercel 자동 제공

### 5.3 SEO & 메타데이터
- title, description, og:image 설정
- 구조화 데이터 (Organization schema)
- sitemap.xml, robots.txt

### 5.4 분석
- Google Analytics 4
- (선택) Meta Pixel — 향후 광고 시 리타겟팅 용도

- **AI 콘텐츠 제작 역량**: AI 숏폼, AI 모델 활용 등 타 에이전시에 없는 고유 역량

## 8. 개발 우선순위 & 마일스톤
